init_knot_vector

PURPOSE

INIT_KNOT_VECTOR creates a knot vector whose entries are distributed

SYNOPSIS

function [ k ] = init_knot_vector( nopx, scale, p )

DESCRIPTION 

 INIT_KNOT_VECTOR creates a knot vector whose entries are distributed
 equally over a Cartisian image grid.
 nopx: number of pixels
 scale: density of knots
 p: polynomial degree
 k: knot vector
